HTML

  • You can use LI while using a UL or OL
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • When adding a source for an image; you place the location after the M and before the greater than sign
  • using ALT you can determine the message provided when a browser cant load a picture

CSS

  • Declarations dictate what the "value" of a given property will be
  • CSS works with HTML to bring color and variety to your document or webpage
  • selector determines the group that will be affected by your declarations

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • "git add -A" stages changes you've made
  • "git commit -m" allows you to commit changes that have been staged with an included message
  • "git pull origin main" syncs our local branch with the remote branch on Github
  • "git push origin [branch-name]" Lets us push our local repository onto Github

JavaScript

  • Control flow is the order in which a computer executes code in a script
  • Defining a function and calling a function are two different but necessary things and are as such written differently
  • A function is written as "Function Function-name()"
  • After defining a function, use curly brackets to specify what its meant to accomplish
  • Calling a function is simply writting the fuction as Function-name()
  • A variable is a named container that allows us to store data in our code.
